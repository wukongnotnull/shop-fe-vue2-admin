# shop-fe-vue2-admin

> 这是一个悟空电商系统管理后台。它只包含了 Element UI & axios & iconfont & permission control & lint，这些搭建后台必要的东西。


## 技术栈
- Vue2.x
- @vue/cli 5.x
- Vue-Router3.x
- Vuex3.x
- Axios1.x
- NProgress
- Element-UI2.x
- WebStorm
- Apifox

## Build Setup

```bash
# 克隆项目
git clone https://github.com/wukongnotnull/shop-fe-vue2-admin.git

# 进入项目目录
cd shop-fe-vue2-admin

# 安装依赖
npm install

# 启动服务
npm run serve
```

浏览器访问 [http://localhost:9528](http://localhost:9528)

## 发布

```bash
# 构建测试环境
npm run build:stage

# 构建生产环境
npm run build:prod
```

## 其它

# 预览发布环境效果

# 预览发布环境效果 + 静态资源分析
